Dijkstra’s algorithm in the web browser with OpenStreetMap

This page demonstrates a JavaScript implementation of Dijkstra’s algorithm applied to geospatial data from OpenStreetMap.

Technologies used:
  • OpenStreetMap geospatial data
  • Leaflet rendering
  • MapBox tiles
  • React & Redux user interface

The shortest path between two random nodes is rendered below. Move your cursor over the map to change the position of the start node.
